﻿﻿﻿﻿﻿# 飞书CLI笔记智能体（NoteAgent）需求文档

## 1. 项目背景与核心目标

### 1.1 项目背景
我们需要开发一个名为“笔记智能体 (NoteAgent)”的系统。该系统基于Python后端和LangChain框架，通过飞书CLI（lark-cli）与飞书生态（云文档、多维表格、消息等）深度连接。用户端目前采用本地命令行交互界面（CLI），并通过Windows .bat批处理文件一键启动。

### 1.2 核心目标
1. **智能笔记管理**：通过自然语言指令创建、更新、搜索和整理飞书笔记
2. **知识检索增强**：基于RAG技术实现文档语义检索和问答
3. **资源自动整理**：围绕笔记主题自动搜集和整理飞书生态中的相关资源
4. **智能体自主规划**：实现多步骤任务拆解和自主执行能力

### 1.3 核心开发原则
⚠️ **大模型与智能体（Agent）的逻辑工作量必须占到整个项目开发工作量的50%以上**。连接层、脚手架和工具链应尽量简化，将核心复杂度聚焦在LangChain的RAG、Tool Calling和Agent状态机上。

## 2. 架构设计：五大核心模块拆分

### 2.1 模块一：Agent核心大脑模块 (Agent Core Brain)

#### 2.1.1 职责
负责任务拆解、意图识别、路由分发和上下文管理。

#### 2.1.2 技术栈
LangChain (ChatModels, AgentExecutor 或 LangGraph)

#### 2.1.3 核心功能
1. **智能体构建**：构建ReAct或OpenAI Functions架构的智能体
2. **会话上下文管理**：管理用户会话上下文（Memory），支持跨轮次笔记讨论
3. **动态工具调用**：根据用户输入（如“帮我把昨天的会议纪要整理成周报”），动态调用飞书工具链

### 2.2 模块二：知识检索与RAG模块 (Knowledge & RAG Service)

#### 2.2.1 职责
负责飞书笔记/文档的向量化、存储、检索和问答。

#### 2.2.2 技术栈
LangChain Embeddings, VectorStores (如 Chroma/Faiss), TextSplitters

#### 2.2.3 核心功能
1. **辅助笔记问答**：对飞书云文档内容进行切片、向量化，实现基于单篇或多篇文档的精准语义检索（QA）
2. **上下文增强**：为Agent提供背景知识注入

### 2.3 模块三：飞书CLI工具箱模块 (Lark CLI Tools)

#### 2.3.1 职责
将飞书CLI（lark-cli）的能力封装为LangChain标准的BaseTool，供Agent自由调用。

#### 2.3.2 技术栈
Python subprocess模块，LangChain @tool装饰器

#### 2.3.3 核心功能
1. **CLI命令封装**：封装飞书CLI核心命令（参照官方larksuite/cli规范）
2. **辅助笔记记录**：封装创建云文档、更新正文、追加内容的Tool
3. **资源整理**：封装搜索相关群聊、提取妙记逐字稿、读写多维表格、搜索知识库的Tool，用于围绕笔记主题自动搜集和整理周边资源

### 2.4 模块四：CLI交互与启动模块 (User Interface & Launcher)

#### 2.4.1 职责
处理用户在终端的输入输出，并提供一键启动脚本。

#### 2.4.2 核心功能
1. **Python交互端** (cli_main.py)：一个精美的命令行循环（Loop），支持流式输出（Streaming）大模型回复，提供清晰的Prompt提示符
2. **Windows启动脚本** (start.bat)：自动检测Python环境、检查飞书CLI登录状态（执行lark-cli auth login校验），并启动主程序

### 2.5 模块五：配置与基础设施模块 (Config & Infra)

#### 2.5.1 职责
管理环境变量、大模型API密钥、日志及飞书权限异常处理。

#### 2.5.2 核心功能
1. **配置管理**：基于pydantic-settings或python-dotenv管理LLM密钥
2. **异常处理**：针对飞书CLI权限不足（如Token过期）提供优雅的错误捕获，引导用户重新授权

#### 5.2.2 Agent工具注册机制
```python
# brain.py - Agent核心
from langchain.agents import AgentExecutor, create_openai_tools_agent
from langchain.prompts import ChatPromptTemplate, MessagesPlaceholder

class NoteAgent:
    def __init__(self):
        self.tools = self._load_tools()
        self.prompt = self._create_prompt()
        self.agent = self._create_agent()
    
    def _load_tools(self):
        """加载所有工具"""
        from src.tools.note_tools import create_document, update_document
        from src.tools.res_tools import search_messages
        from src.rag.vector_store import search_documents
        
        return [create_document, update_document, search_messages, search_documents]
    
    def _create_prompt(self):
        """创建提示词模板"""
        return ChatPromptTemplate.from_messages([
            ("system", "你是一个智能笔记助手..."),
            MessagesPlaceholder(variable_name="chat_history", optional=True),
            ("human", "{input}"),
            MessagesPlaceholder(variable_name="agent_scratchpad")
        ])
    
    def _create_agent(self):
        """创建智能体"""
        return create_openai_tools_agent(
            llm=self._get_llm(),
            tools=self.tools,
            prompt=self.prompt
        )
    
    def run(self, user_input: str) -> str:
        """执行用户指令"""
        executor = AgentExecutor(agent=self.agent, tools=self.tools, verbose=True)
        return executor.invoke({"input": user_input})
```
